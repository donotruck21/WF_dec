What are databases?

A structured set of data

Databases are a collection of information. They are used for creating, manipulating, and deleting data. There are many different types of databases and they can be classified in a basic approach of relational database or a non-relational database. Relationships means a set of our data is related to another set of data. Meanwhile in a non-relational database, there is not a related common setion of data. Our adta bases are put into a section of data vcalled schemas, inside of each schema each dataset is organized into tables. Those tables are organized into rows and columns.
