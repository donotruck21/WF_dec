Javascript Basics

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language. Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

  • Variable Declaration Javascript variables are containers for storing data values - imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All Javascript variables must be identified with unique names. These unique names are called identifiers. var x
  • Variable Assignment Assignment operators assign variables to Javascript variables.- our cup can now hhave coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a value to a variable. var x = 10
  • Data Types Data typrs are an important concept; to be able to operate on variables you need to know the data type. There are six data types that are Javascript primitives: Boolean - true or false;null - null aka nothing; Number - 42 or 3.14159;String - Coding Dojo Rocks!; Array - [1, "Coding","Dojo"] and Object {first_name:"Jane", last_name:"Doe"};
  • Conditionals When you write code, you want to perform different operations for different decisions - hitting different code blocks based on values or conditionals that have been met. You can use conditional statements in your code to accomplish this. These are the following condtional staments: if is a specified condition is true, do this code in our code block; else if to specify a new conditon to test, if the first condition is false; else we excute tis block of code;
  • Loops There are many different kinds of loops in every programming language, but they all essentially do the same thing: they will repeat an actionsome number of times. Imagine you have to run a mile, around the track four times and then you stop. That's a loop!
